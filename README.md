# Smart-Job-Scraper
AI-powered job aggregation platform that automatically scrapes and consolidates job listings from multiple job portals. Built with Python, FastAPI, asynchronous web scraping, and REST APIs to provide real-time job search and filtering capabilities.


---

## Features

- Multi-source job scraping
- FastAPI REST API
- Interactive Swagger API documentation
- Job search and filtering
- Structured JSON responses
- Frontend dashboard
- High-performance asynchronous backend
- Easy deployment with Docker or cloud platforms
- Modular and scalable architecture
